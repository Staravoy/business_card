Досвід та проекти

1. Веб-додаток для контролю за транспортними засобами

Ініціював, розробив та впровадив веб-додаток для обліку та контролю за транспортними засобами (далі – ТЗ) підрозділів України, що підвищило ефективність обліку та задіяння ТЗ.

Функціонал додатку

Додаток реалізовано за допомогою фреймворку Django мови програмування Python та включає наступний функціонал:

  • Авторизація з різним рівнем доступу;
  • Отримання робочого кабінету різного рівня доступу;
  • Кабінет користувача (старшого по ТЗ в підрозділі): можливість заповнювати дані про ТЗ, вести особисті записи, контролювати задіяння ТЗ одного підрозділу;
  • Кабінет керівника: зібрана детальна інформація з усіх підрозділів по кожному ТЗ з можливістю фільтрації та виведення даних у таблиці;

Інформація про базу даних та користувачів

  • База даних містить приблизно 2000 одиниць ТЗ різного типу;
  • Кількість користувачів — до 100.

Посилання на GitHub репозиторій: https://github.com/Staravoy/PopPortal

2. Додаток для введення поправок при стрільбі

Придумав та написав додаток для введення поправок при стрільбі зі зброї з оптичним прицілом.

Функціонал додатку

Додаток реалізовано за допомогою мови програмування JavaScript та бібліотеки React і включає наступний функціонал:

  • Можливість введення даних при яких ведеться стрільба;
  • На основі розрахунків визначаються поправки;

Цей додаток сприяв підвищенню рівня самостійної підготовки стрільців як фахівців, так і менш досвідчених.

Посилання на GitHub репозиторій: https://github.com/Staravoy/moa

Посилання на онлайн-версію: https://staravoy.github.io/moa/

3. Локальний додаток для автоматизації робочих процесів

Розроблено локальний додаток для автоматизації процесу збору даних про особовий склад (о/с) підрозділів.

Метод реалізації

Додаток реалізовано за допомогою мови програмування Python та бібліотеки pandas.

Функціонал додатку

Основний функціонал додатку:

  • Автоматичне збирання даних щодо перебування о/c підроздів за вказаний період;
  • Обробка даних та створення нової таблиці з відфільтрованою інформацією;
  • Збереження зібраної таблиці у вказаному місці;
  • Швидкість виконання: приблизно 10 хвилин замість 6 годин вручну.

Посилання на GitHub репозиторій: https://github.com/Staravoy/count_reiting

4. Десктопний додаток Конвертер валют

Маленький додаток для розрахунку вартості валюти з доларів США та Євро у Гривню.

Метод реалізації

Додаток реалізовано за допомогою мови програмування Python та бібліотеки Beautiful Soup.

Особливість додатку

Додаток здійснює збір інформації з офіційного сайту Міністерства фінансів України, де відбирається вартість валюти у різних банках України, вираховується середнє значення і на основі нього здійснюються розрахунки вартості валюти для користувача.

Функціонал додатку

  • Автоматичний збір курсів валют з офіційних джерел;
  • Розрахунок вартості долара та євро в гривнях на основі середнього курсу;
  • Простий та інтуїтивно зрозумілий інтерфейс;
  • Швидкість обчислень та оновлення курсу валют.

Посилання на GitHub репозиторій: https://github.com/Staravoy/currency_converter

5. Веб-додаток: Журнал спостережень

Додаток реалізовано за допомогою Python та Django. Дозволяє користувачу вести власний журнал досліджень з будь-якої теми, що він сам додасть.

Функціонал додатку

  • Реєстрація користувача;
  • Вхід користувача через логін та пароль;
  • Можливість додавати тему дослідження;
  • Можливість додавати запис до теми дослідження з датою та часом введення;
  • Кожен користувач бачить тільки свої дослідження.

Посилання на GitHub репозиторій: https://github.com/Staravoy/learning_log
